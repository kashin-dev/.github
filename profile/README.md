 Kashin

 Kashin is a Peruvian fintech building accessible microlending infrastructure — regulated by the SBS, operating since 2020.

 We develop the full stack in-house: mobile app, backend services, risk models, and internal tooling. Our systems process real money for real people, so we take
 reliability, correctness, and security seriously.

 ### What we build

 - Mobile — React Native app for loan origination and wallet management
 - Backend — Distributed services with event-driven architecture (RabbitMQ, MongoDB)
 - Risk — ML models for credit scoring and fraud detection
 - Infrastructure — High-performance components in Rust where it counts

 ### How we work

 We ship in small, frequent iterations. Code that touches money gets double review. Nothing goes to production without passing through staging first.

 ### Values

 We believe financial services should be reliable above all else. We'd rather ship less and sleep well than move fast and break trust.
